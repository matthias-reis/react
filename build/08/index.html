Create an event system

Variant 2 is a little bit more complex, but easier to handle in slightly bigger scenarios

  • Create a global Event bus (having functions like trigger() or on())
  • Pass data to these Events like trigger(ADD_HERO, {name: 'Hawkeye'})
  • Trigger it in the action holding component (like the Button)
  • Consume it in the stateful component (like the HeroList)
