Render, Components and JSX

Get markup onto your page

How to start

  1. React.render({jsx}, {dom-node}) as a starting point
  2. Create a ES6 class that extends from React.Component
    or use React.createClass()
  3. Create a render() method in that class
  4. Use the component as a custom JSX tag like <ComponentName/>

JSX
