Handing over callback functions

The main design goal for React applications is to find the right way to pass the data to your components. Part of this presentation are two easy methods. The ideal and surely more complex structure that suits our needs is to be defined afterwards

First let's start with Variant 1: Callback handover

  • Props don't necessarily just transport data chunks to our components
  • Functions are first class citizens in JS and so they can be passed down, too.
  • Solution:
    • Define a function in a state holding Component like HeroList
    • Pass it as a prop to the Button component <Button label="Add Superhero" callback={handleAdd}>
