Props

Create children and inherit properties

this.props are the second way to apply data to components:

  • Just add a HTML-like property to the component's declaration
  • Like <App name="List Application"/>
  • This can be of any type that Javascript supports - even a function. In this case, you must use curly braces.
  • Like <Counter number={42}/> or <Button callback={this.onButtonClick}/>
  • You can use the props in the child component by using this.props.propertyName
  • props are immutable. They should never be changed by their component
  • the dynamic comes from the outer element

Combine State and Props

  • You may hand over parts of the state - e.g. of the uppermost component - to its children
  • Like in an App component <SideNavigation items={this.state.items}/>
  • And by altering the state this.setState({items: [...this.state.items, myNewNavItem]}), you can control the behavior of the child from its parent.
  • But beware: Think of reusability. A property should only obtain the properties that it really needs.
